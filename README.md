# Simple Calculator

A clean and modern web-based calculator built with HTML, CSS, and JavaScript.

## Features

- **Basic Operations**: Addition, subtraction, multiplication, and division
- **Clear Functions**: All Clear (AC) and Delete (DE) buttons
- **Responsive Design**: Works on both desktop and mobile devices
- **Modern UI**: Dark theme with gold operator buttons and smooth hover effects
- **Decimal Support**: Handles decimal point calculations

## Technologies Used

- **HTML5**: Structure and layout
- **CSS3**: Styling with modern effects and animations
- **JavaScript**: Basic functionality for calculations
